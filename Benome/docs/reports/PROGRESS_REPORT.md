## 🔧 技术细节

### 数据库
- 使用 SQLite (`benome.sqlite`)
- 通过 Alembic 管理迁移
- 无需新增表，使用现有 `booking` 表

### API 设计
- RESTful 风格
- 统一响应格式
- 错误码规范：
  - 200: 成功
  - 400: 请求错误
  - 401: 未授权
  - 403: 禁止访问
  - 404: 不存在
  - 409: 冲突

### 前端技术
- 原生 HTML/CSS/JavaScript
- 无框架依赖
- 响应式设计
- 渐进增强
