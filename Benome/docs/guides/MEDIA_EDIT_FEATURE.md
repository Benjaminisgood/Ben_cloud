# Benome 媒体编辑功能使用说明

## 🎉 功能更新

**2026-03-02 09:35** - 新增媒体编辑功能，现在可以修改已上传媒体的信息！

## ✏️ 编辑功能

### 可以修改的内容

- 📝 **标题** - 修改媒体文件的标题
- 📄 **描述** - 更新详细描述
- 📸 **封面状态** - 设置或取消封面

### 使用步骤

1. **进入媒体管理页面**
   ```
   http://your-domain:8200/admin/properties/{房间 ID}/media
   ```

2. **点击"编辑"按钮**
   - 在任意媒体项目右侧找到"编辑"按钮
   - 点击后弹出编辑模态框

3. **修改信息**
   - 在模态框中修改标题、描述
   - 勾选/取消"设为封面"
   - 点击"保存修改"

4. **查看结果**
   - 页面自动刷新
   - 显示更新后的信息

## 🔧 技术细节

### API 端点

```http
PATCH /api/media/{media_id}
Content-Type: application/json

{
  "title": "新标题",
  "description": "新描述",
  "is_cover": true
}
```

### 请求参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| title | string | 否 | 媒体标题（最大 200 字符） |
| description | string | 否 | 媒体描述 |
| is_cover | boolean | 否 | 是否设为封面 |
| sort_order | integer | 否 | 排序顺序 |
| is_active | boolean | 否 | 是否启用 |

### 权限要求

- ✅ 需要 **admin** 角色
- ✅ 需要登录状态
- ✅ SSO 认证

## 📋 使用场景

### 场景 1：添加缺失的信息

上传时忘记填写标题和描述？
1. 点击"编辑"
2. 补充信息
3. 保存

### 场景 2：修改错误信息

标题写错了？描述有误？
1. 点击"编辑"
2. 修正内容
3. 保存

### 场景 3：更换封面

想换一张图片作为房间封面？
1. 找到想设为封面的图片
2. 点击"编辑"
3. 勾选"设为封面"
4. 保存

或者直接在列表中点击"设为封面"按钮（如果当前不是封面）。

### 场景 4：批量更新封面

为钱塘云湾中心民宿设置宣传视频为封面：
1. 找到宣传视频
2. 点击"编辑"
3. 勾选"设为封面"
4. 保存

现在视频会显示在房间详情页最前面！

## ⚠️ 注意事项

### 不能修改的内容

以下信息上传后**无法修改**：
- ❌ 媒体文件本身（需要删除后重新上传）
- ❌ 文件类型（图片/视频）
- ❌ 文件大小
- ❌ OSS 存储路径

### 封面规则

- ✅ 每个房间只能有一个封面
- ✅ 设置新封面时，原封面自动取消
- ✅ 图片和视频都可以设为封面
- ✅ 建议将最佳展示内容设为封面

### 删除警告

- ⚠️ 删除操作**不可恢复**
- ⚠️ 删除会同时删除 OSS 上的文件
- ⚠️ 请谨慎操作

**推荐作为封面的内容：**
- 🏆 最佳角度的房间全景
- 🎬 官方宣传视频（如果有）
- 🌟 最能吸引客人的特色区域

**不推荐作为封面的内容：**
- ❌ 卫生间照片
- ❌ 局部细节图
- ❌ 光线不足的照片

## 🐛 故障排查

### 问题：点击"编辑"没反应

**解决方法：**
1. 刷新页面（Ctrl+F5）
2. 检查浏览器控制台是否有错误
3. 清除浏览器缓存

### 问题：保存失败

**可能原因：**
- 登录已过期
- 权限不足（需要 admin 角色）
- 网络连接问题

**解决方法：**
1. 重新登录
2. 确认账号有 admin 权限
3. 检查网络连接
4. 查看服务日志：`tail -f /root/Ben_cloud/Benome/logs/benome.log`

### 问题：修改后没有立即显示

**解决方法：**
- 页面会自动刷新，如未刷新请手动刷新
- 清除浏览器缓存
- 检查是否保存成功（看 flash 消息）
